Principais Formatações

Negrito / Destaque

Nesta frase temos um termo em negrito usando a tag <b> essa é a forma não semântica.

Nesta frase temos um termo em destaque usando a tag <strong>, essa é a forma semântica.

Itálico / Ênfase

Nesta frase temos um termo em itálico, usando a tag <i>, essa é a forma não semântica.

Nesta frase temos um termo em itálico, usando a tag <em>, essa é a forma semântica.

Texto Marcado

Nesta frase temos um termo marcado, usando a tag <mark>, essa é a forma semântica.

Nesta frase temos um termo marcado em outra cor, usando a tag <mark> e as CSS.

Texto Grande e Pequeno

Nesta frase temos um termo grande e um termo pequeno, usando a tag <big> que está obsoleta e usando a tag <small> que não está obsoleta.

Texto Deletado

Nesta frase temos um termo deletado, que deve ser lido mas desconsiderado, usando a tag <del>.

Texto Inserido

Nesta frase temos um termo inserido, para dar ênfase nele e indicar que foi adicionado depois, não apenas pelo sublinhado, usando a tag <ins>.

Temos também a tag <u>, que também deixa, sublinhado, porém, essa é a forma não semântica.

Texto Sobrescrito

Nesta frase temos um termo sobrescrito, usando a tag <sup>.

Texto Subscrito

Nesta frase temos um termo subscrito, usando a tag <sub>.
